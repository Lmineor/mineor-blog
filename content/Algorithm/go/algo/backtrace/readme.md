# 回溯算法

* 回溯搜索的遍历过程

回溯法一般在集合中递归搜索，集合的大小构成了树的宽度，递归的深度构成了树的深度。

回溯函数伪代码遍历过程如下：

```bash
for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
    处理节点;
    backtracking(路径，选择列表); // 递归
    回溯，撤销处理结果
}
```

for循环就是遍历集合区间，可以理解一个节点有多少个孩子，这个for循环执行多少次。

算法模板框架为：

```c
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
